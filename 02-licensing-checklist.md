# 2. Licensing checklist (phase by phase)

A master checklist you can adapt to any of the three regimes. Work top-to-bottom;
don't start the application until Phase 1 is genuinely done.

## Phase 0 — Strategy & feasibility
- [ ] Define the exact regulated activities you will perform (exchange, custody, broker-dealer, OTC, etc.)
- [ ] Pick the jurisdiction against market, capital, custody and token scope
- [ ] Confirm minimum capital and that you can fund it (paid-up, not pledged)
- [ ] Map the corporate structure and ultimate beneficial owners (UBOs) — no nominees hiding control
- [ ] Budget for advisers, capital, and a 6–12 month timeline

## Phase 1 — Foundations (pre-application)
- [ ] Incorporate the licensed entity in the right jurisdiction
- [ ] Draft a credible regulatory business plan (products, clients, geographies, volumes)
- [ ] Identify and vet fit-and-proper key persons (MLRO, Compliance Officer, CISO, DPO, directors)
- [ ] Enterprise-Wide Risk Assessment (EWRA) completed
- [ ] Core policies drafted: AML/CFT, sanctions, KYC/CDD/EDD, Travel Rule, conflicts, complaints, outsourcing, BCP
- [ ] Technology & security architecture documented (custody model, key management, monitoring tooling)

## Phase 2 — Application
- [ ] Complete the regulator's application forms and pay fees
- [ ] Submit business plan, financials, org chart, UBO/source-of-funds evidence
- [ ] Submit the policy suite + EWRA + governance map
- [ ] Submit key-persons applications (fit-and-proper questionnaires)
- [ ] Track submission references and acknowledgements

## Phase 3 — Regulator engagement (RFIs)
- [ ] Stand up a single owner and a tracker for Requests for Information (see [04](04-rfi-response-framework.md))
- [ ] Answer each RFI precisely, with evidence, against the right rule reference
- [ ] Remediate any gaps the regulator flags; re-submit cleanly

## Phase 4 — Pre-launch / conditions
- [ ] Satisfy any conditions attached to the in-principle approval
- [ ] Independent audit / readiness review of controls
- [ ] Onboard transaction-monitoring, screening and Travel Rule tooling in production
- [ ] Staff trained; testing evidence retained

## Phase 5 — Go-live & ongoing
- [ ] Licence granted; commence regulated activity within scope only
- [ ] Ongoing reporting, periodic EWRA refresh, independent AML audit
- [ ] Maintain fit-and-proper status of key persons; notify material changes
