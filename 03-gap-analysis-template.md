# 3. Gap-analysis template

Before you apply, assess yourself against the rulebook exactly as an examiner
would. A gap analysis converts a long rulebook into a prioritised action list.

## How to run it
1. Extract every requirement from the rulebook into rows (one obligation per row).
2. For each, record what you **have**, what's **missing**, and a **severity**.
3. Assign an owner and a due date. Close criticals before applying.

## Severity scale
- **CRITICAL** — missing; blocks the application or a core control.
- **HIGH** — partially in place; examiner will challenge it.
- **MEDIUM** — exists but needs improvement / evidence.
- **LOW** — minor / cosmetic.
